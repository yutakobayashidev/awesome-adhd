# Tiimo

Tiimoは、ADHDや自閉スペクトラム症の人、また柔軟な構造を必要とする人に向けた視覚的な計画アプリ。公式ページでは、視覚的な予定表、集中タイマー、やることリスト、ウィジェット、AIによるタスク分解、気分記録などを組み合わせて、[[executive-function]]を外から支える道具として説明されている。

## 何を支える道具か

- **時間の見える化**: カウントダウンや視覚的なタイムラインにより、[[time-management]]の負荷を下げる。
- **次の行動の見える化**: ホーム画面・ロック画面・Apple Watchなどのウィジェットで、アプリを開かなくても次の予定やタイマーを確認できる。これは[[external-memory]]として働く。
- **着手の補助**: AIが入力されたタスクをステップ化し、所要時間を見積もり、スケジュールへ落とし込むと説明されている。これは[[task-initiation]]や[[work-routines]]と相性がよい。
- **環境への埋め込み**: 複数端末同期、色・アイコン・レイアウトの変更により、本人の記憶や意志だけでなく、日常の表示環境へ予定を置く[[environment-design]]として使える。

## 注意点

誤投入側から移した Tiimo neurodivergent planner source は、日本語ページの neurodivergent 向け説明を補強する。視覚的予定、集中タイマー、widget / live activity、気分記録、AI による task 分解・見積もりを、[[executive-function]] の外部化として読む。ただし製品ページなので効果検証ではない。^[raw/articles/tiimo-neurodivergent-planner-2026.md]


このページはTiimo自身による製品説明であり、独立した効果検証ではない。ADHDや自閉スペクトラム症への「効く／治る」道具としてではなく、予定・時間・行動を見える形にして実行機能を補助する候補として扱う。
