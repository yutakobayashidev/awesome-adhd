# Body doubling / life admin sources for Admin Night

Admin Night という語そのものは Web 検索ではまだ強い独立語としては確認しにくい。一方で、投稿本文が指す「友達と集まり、請求書、確定申告、細かい事務を一緒に片づける」行為は、既存の body doubling / life admin / coworking 文脈と近い。

## ADDA: The ADHD Body Double

ADDA は body doubling を、ADHD の人が「もう一人がそばにいる状態」で、嫌だったり退屈だったりする仕事を終えるための生産性戦略として説明する。body double は本人の作業を代行するのではなく、集中を保ち、気が散ることを減らし、動機づけを増やす役割を持つ。例として、洗濯、請求書の支払い、運動、仕事の project が挙がる。本文は、科学的な説明は未確立だが、多くの実践者にとって有用な手法として扱う。

Source: https://add.org/the-body-double/

## ADDitude: Get More Done with a Body Double

ADDitude は、ADHD の人は housework、homework、bill paying、その他の task を、一緒にいる人がいると集中しやすいと説明する。特に「paperwork に遅れがち」「請求書の支払い、保険書類、その他 financial tasks が難しい」時に body double が助けになる、とする。body double は助言や監督ではなく、そばにいてくれる存在として説明されている。

Source: https://www.additudemag.com/getting-stuff-done-easier-with-a-friend-body-double/

## Cleveland Clinic: Body Doubling and ADHD

Cleveland Clinic は body doubling を、ADHD の人が誰かと並んで作業することで、focus、accountability、motivation を助ける方法として説明する。Michael Manos, PhD は、これは external executive functioning に近く、本人の外側に実行機能を置くようなものだと述べる。20〜90分程度の session、友人・家族・同僚・オンライン group での実践、作業後の休憩や雑談が挙げられている。

Source: https://health.clevelandclinic.org/body-doubling-for-adhd

## Admin Night への読み替え

Admin Night は、body doubling を「生活の細かい事務」に特化し、しかも event 化・social 化したものとして読める。単なる作業効率ではなく、以下の変換が起きている。

- 孤独な義務 → 友達との予定
- 先延ばししがちな事務 → 時間枠のある session
- 恥や不安を伴う money / tax / paperwork → 互いに片づける共有場
- 意志力依存 → 外部化された実行機能と説明責任
- boring task → 軽い社交・儀式・報酬を含む event

注意点として、金銭・税・収入差を扱うため、プライバシー境界、同席者の選び方、見せる情報の範囲、互いに踏み込まない規約が重要になる。
