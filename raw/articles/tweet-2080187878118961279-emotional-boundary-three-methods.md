# 感情をもらいすぎない3つの方法

Author: ゆの｜HSP×ADHD×ASDメンタル戦略 (@yuno_mind_lab)
Created: Thu Jul 23 07:06:52 +0000 2026
Likes: 0, Retweets: 0, Replies: 0

## Tweet Text

感情をもらいすぎん3つ。

①拾った瞬間「これは誰の感情？」って一回聞く、ラベルを貼るだけで流入が減る
②しんどい場面では物理的に1歩離れる、距離は防波堤になる
③帰ったら「今日拾った分」を紙に書いて返却する
全員救おうとせんでいい。まず自分の感情の席を自分に返してあげての。

## Analysis

ADHDに多い感情調整困難（他者の感情を過剰に受け取る／共感疲労）に対する、具体的かつ物理的な3段階の対処法。①認知的ラベリング（メタ認知）、②物理的距離（環境調整）、③外在化と「返却」の儀式（筆記による外在化＋心理的境界の再設定）。すべて「意志で耐える」ではなく「仕組みと行動で境界を作る」アプローチ。
