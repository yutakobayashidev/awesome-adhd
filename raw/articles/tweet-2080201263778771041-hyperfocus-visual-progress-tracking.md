# 過集中を「見える化」と数字記録で呼び起こす

Author: 凜｜人生を科学する心理学 (@aibijo_aoi)
Created: Thu Jul 23 08:00:04 +0000 2026
Likes: 0, Retweets: 1, Replies: 1

## Tweet Text

よく聞いて。世の中で天才って言われる人、ほぼ全員「1つのことに異常に没頭できる人」。それADHDの過集中と同じ仕組みなんだよ。つまりお前が欠点だと思ってるその集中力、磨けば最強の武器になる。眠らせてるだけ。起こし方は簡単で、まず自分の改善を「見える化」する。体重でも貯金でも作業時間でも、数字で記録して伸びていくグラフを毎日眺める。ADHDの脳は「変化」と「達成」に異常に反応するから、伸びてるのが見えた瞬間スイッチが入る。あとはその快感を追いかけるだけで勝手に過集中。武器はもう持ってる。向ける先を変えるだけ。これ知ってるだけで全然違う。

## Analysis

過集中を「意志で呼び起こす」のではなく、数値の可視化によるドーパミンフィードバックで自然に誘発するという具体的戦略。体重・貯金・作業時間など、定量化できるものをグラフ化して「変化」と「達成」を視覚的に確認することで、ADHD脳の報酬系を外在的に刺激する。パワー系ソリューションとしての「見える化」の応用。
