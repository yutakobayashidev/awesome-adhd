# Fear of Missing Out

Fear of Missing Out（FoMO / FOMO）は、自分がいない場所で他者が価値ある経験をしているかもしれないという不安と、それを確かめるためにつながり続けたい欲求を指す。Przybylski らはこの語を、SNS が「参加できる社会的選択肢」を実際に追える量より多く見せることで生じる緊張として定義し、FoMO Scale を作った。^[raw/papers/przybylski-2013-fomo-scale.md]

ADHD Wiki で重要なのは、FoMO を「意思が弱い」「SNS依存」と短絡しないこと。Elhai らの総説は、FoMO が SNS、スマホ、インターネットの多い利用、問題化した利用、不安、抑うつ、否定的感情、生活満足の低さと関連すると整理している。一方で、多くの研究は横断・自己報告であり、因果方向はまだ弱い。^[raw/papers/elhai-2021-fomo-overview.md]

## ADHD 支援への含意

- **禁止より有限化**: Fitz らの通知実験では、通知を 1 日 3 回にまとめる条件は注意、生産性、気分、統制感を改善した。一方、通知を完全になくす条件では不安と FoMO が高かった。通知・SNS・メールを「全部見せない」方向へ寄せすぎると、安心ではなく確認衝動を増やす可能性がある。^[raw/papers/fitz-2019-notification-batching-fomo.md]
- **終了手掛かりを作る**: FoMO は「まだ何かあるかもしれない」という未確定性で強まりやすい。`今回はここまで`、残件数、次の確認時刻、確認済みの範囲は、情報を断つためではなく、未確認範囲を小さくするための [[digital-interruptions]] 対策になる。
- **責める文言を避ける**: FoMO が強い人に「また見た」「戻れなかった」という失敗表示を出すと、自己評価の低下や回避を増やしうる。[[emotion-regulation]] と [[task-resumption]] の支援では、延長・保留・次回確認を人格評価にしない。
- **自己報告だけで測らない**: Akbari らの systematic review / meta-analysis は、FoMO と internet use の関連は条件で幅があり、双方向性も十分には分かっていないとする。支援の評価では FoMO 尺度だけでなく、予定超過、アプリの迂回、作業復帰、次の一手の保持を合わせて見る。^[raw/papers/akbari-2021-fomo-internet-use-meta-analysis.md]

## 認知負荷との接続

Montag と Markett の大規模横断研究では、SNS 非利用者は FoMO と日常の認知的失敗が低く、SNS 利用者では FoMO、Social Networks Use Disorder 傾向、Cognitive Failure Questionnaire の得点が関連した。特に online 文脈の state FoMO と問題化した SNS 利用の関連が強かった。ただし横断研究なので、FoMO が認知的失敗を起こすと断定はできない。^[raw/papers/montag-2023-fomo-cognitive-failure.md]

このため、ADHD 支援では「FoMO を消す」より、いつ確認欲求が強まり、どの提示なら戻りやすく、どの隠し方なら逆に不安が増えるかを個人ごとに見る方がよい。[[attention-control]]、[[task-resumption]]、[[external-memory]] とつなげて、通知の量ではなく、本人の復帰・納得・安心を観察する。

## 長期的な因果はまだ弱い

Groenestein らの 3 波縦断研究は、FoMO と SNS 利用、心理的欲求充足、well-being の関係を between-person と within-person に分けて調べた。個人間では変数が関連したが、個人内では一部の相互経路が見られたに留まり、心理的欲求充足や well-being の低さから FoMO へ流れるという一般的な説明には疑問も残した。^[raw/papers/groenestein-2024-fomo-social-media-longitudinal.md]

したがって、FoMO 支援は「SNS を使うから悪い」「FoMO があるから使う」のどちらかに固定せず、状態、場面、確認できる範囲、次回確認の約束を分けて設計する必要がある。
