# ADHD子育ての工夫

## 概要

ADHDのある子どもを育てる親たちから集められた「やってよかった」実践知のまとめ。専門的介入ではなく、家庭で取り入れられる環境・生活・勉強・コミュニケーション面の工夫が中心。いずれも「子どもの意志に頼らず、環境や仕組みで支える」発想に立つ。

以下の内容は @heyasupo 氏が親御たちから収集した41項目をカテゴリ別に整理したもの。

## 環境・生活の工夫

- 同じ靴下を複数買いして「ペア探し」の概念をなくす → [[environment-design]]
- 重めの布団（ウェイトブランケット）を使う → [[sleep]]
- トランポリンやバランスボールを家に置く（感覚刺激・運動の出口）
- タイムタイマー（残り時間が色でわかる時計）の導入 → [[time-management]]
- 部屋の視覚的ノイズを減らす（断捨離） → [[environment-design]]
- 物の定位置を写真やテプラで明示する → [[external-memory]]
- 翌日の準備ボックスを玄関に置く → [[forgetfulness-countermeasures]]
- GPSタグの活用（忘れ物・迷子防止） → [[external-memory]]
- イヤーマフやノイズキャンセリングの活用 → [[attention-control]]
- 服のタグを切る・肌触りの良い服を選ぶ（感覚過敏対策）
- 朝食にタンパク質を意識的に摂る
- 寝る前のルーティンを完全固定化する → [[sleep]]
- やるべきことをホワイトボードで可視化する → [[external-memory]]
- リュックの中身をざっくり投げ込み収納にする（整理のハードルを下げる）
- スマートスピーカーで時間を知らせる（声かけの自動化） → [[time-management]]

## 勉強面の工夫

- 宿題中にハンドスピナーやスクイーズなどの手遊びを許容する（刺激欲求を満たしながら集中）
- 宿題をゲーム化・クイズ化する（ドーパミンを活用）
- 勉強時間を10分くらいの細切れにする → [[task-initiation]]
- 集中できる環境を作る（パーテーションなど） → [[attention-control]]
- 提出物専用のクリアファイルを作る → [[forgetfulness-countermeasures]]
- スタンディングデスク（立って勉強）を認める
- 教科書やプリントを拡大コピーする（視覚的処理の補助）

## 声かけ・コミュニケーションの工夫

- 指示は短く・具体的に・一つずつ出す（ワーキングメモリ負荷を下げる） → [[working-memory]]
- パーソナルスペースをフラフープなどで視覚的に教える
- 話す前にまず視線を合わせる（肩に触れる）
- 「ダメ」の代わりに「〜しよう」と肯定形で伝える
- 叱る基準を明確にする（命の危険・他害以外はスルー）
- できたことを実況中継して褒める（即時フィードバック）
- スモールステップで成功体験を積ませる → [[task-initiation]]
- クールダウンできる場所を作る → [[emotion-regulation]]
- 予告をしてから行動を切り替えさせる（切り替えの準備時間を与える）
- 1日1回は必ずハグやスキンシップをする
- 感情的に怒りそうになったら親がその場を離れる（親自身の感情調整）
- 望ましくない行動をあえて「無視」して注目を与えない
- 「助けて」「手伝って」という援助要請のスキルを教える
- 予定変更があるときは、わかった時点で早めに伝える
- 親自身の失敗談を明るく話して「失敗しても大丈夫」と伝える

## 親のマインドセット

- 「普通」や「平均」という物差しを捨てる → [[self-experimentation]]
- 多動を行動力、衝動性を決断力と捉え直す
- 親の会やSNSで仲間を見つける
- 専門家を頼ることを躊躇しない → [[public-support]]
