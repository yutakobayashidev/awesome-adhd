# 展望記憶

展望記憶は、「あとでこれをする」を覚えておき、適切な時点で実行する働き。ADHDの生活では、締切、持ち物、服薬、返信、予約、提出物などがここに関わる。[[working-memory]]と[[time-management]]にまたがる。

Fuermaierらは成人ADHDの複雑な展望記憶を扱った研究として取り込む。Jylkkäらは、仮想の部屋で日常課題を進めるオンライン3D課題を使い、成人ADHDの目標志向行動と展望記憶を評価する試みとして取り込む。^[raw/papers/fuermaier-2013-complex-prospective-memory-adult-adhd.md]^[raw/papers/jylkka-2023-everyday-prospective-memory-adult-adhd.md]

Masicampo & Baumeisterの計画作成研究は、未完了目標が頭に残る効果を、具体的計画で軽減できる可能性を示す隣接研究として読む。ADHD支援では「覚えておく」より「いつ・どこで・何をするか」を外へ出す設計に接続できる。^[raw/papers/masicampo-2011-plan-making-unfulfilled-goals.md]

## 実装上の含意

- [[external-memory]]: 予定・持ち物・返信を頭に保持せず、チェックリスト、カレンダー、場所、通知へ移す。
- [[time-management]]: 「何時にやる」だけでなく、行動のきっかけと場所を決める。
- [[task-resumption]]: 中断後に戻るため、次の一手と再開位置を残す。
